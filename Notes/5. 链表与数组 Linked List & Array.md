# 链表与数组 Linked List & Array
## Linked List的理解
- set a linked list class and print
```py
class ListNode:
    def __init__(self, data):
        self.value = data
        self.next = None
       
class LinkedList:
    def __init__(self, head = None):
        self.head = head

def printLinkedList(self):
    node = self.head
    while node:
        print(node.value)
        node = node.next
        if node:
            print('->')
        
ll = LinkedList()
node1 = ListNode(1)
node2 = ListNode(2)
node3 = ListNode(3)

ll.head = node1
node1.next = node2
node2.next = node3

printLinkedList(ll)
node1 = node2
printLinkedList(ll)

## 结果都是 “1 -> 2 -> 3"
```
1. node 是Linked List里的局部变量还是全局变量？ 局部
   - node1 是一个指针，在32位即中占有4个字节。也可以理解为引用，存的是一个内存的地址。所以一个ListNode占8个字节，val占4个，next占4个。
   - 可以理解为，内存是一个大数组，ref和pointer都是数组的下标，即index。
